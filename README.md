# 인플레이션 환경에서 디저트 소비 선호도 분석 및 ML 예측 모델 비교

5개년 서울시 상권분석 서비스 데이터를 활용하여, 행정동 단위로 **카페·제과점(디저트)** 소비 선호도 변화를 분석하고 머신러닝 기반 예측 모델을 비교합니다.

## 프로젝트 개요

- **분석 대상**: 서울시 상권 데이터 (5개년, 행정동 단위)
- **업종 필터**: 카페, 제과점 (디저트)
- **목표**:
  1. 인플레이션 환경에서 디저트 소비 선호도 변화 분석
  2. 머신러닝 예측 모델 구축 및 비교

## 프로젝트 구조

```
machine_learning/
├── data/                 # 데이터 (raw, processed)
│   ├── raw/              # 원본 서울시 상권 데이터
│   └── processed/        # 전처리된 디저트(카페·제과점) 데이터
├── notebooks/            # 탐색·분석 Jupyter 노트북
├── src/                  # 재사용 가능한 코드
│   ├── data/             # 데이터 로딩, 전처리
│   ├── analysis/         # 분석 유틸
│   └── models/           # 모델 학습·평가
├── outputs/              # 분석 결과, 시각화, 모델 저장
├── requirements.txt
└── README.md
```

## 데이터

- **출처**: 서울시 상권분석 서비스 (5개년)
- **단위**: 행정동
- **업종**: 카페, 제과점

## 사용 방법

```bash
# 가상환경 생성 및 활성화
python -m venv venv
source venv/bin/activate  # Windows: venv\Scripts\activate

# 의존성 설치
pip install -r requirements.txt

# Jupyter 실행
jupyter lab
```

## 분석 흐름

1. **데이터 로딩** – 서울시 상권 데이터 로드
2. **디저트 업종 필터링** – 카페·제과점만 추출
3. **시계열 분석** – 인플레이션과 소비 선호도 변화
4. **모델 구축** – 여러 ML 모델 학습
5. **모델 비교** – 성능 평가 및 비교
