---

### Block 4: 계절·시간 통제

| 변수 | 설명 |
|------|------|
| `month_sin`, `month_cos` | 계절성 |
| `time_index` | 시간 추세 (선택) |
| `C(연도×분기)` | FE 모델용 시간 더미 |

---

### 모델별 Feature Set

**Baseline**
```
X_baseline = [
    lag1_비중,
    lag4_비중,
    성장률,
    month_sin,
    month_cos
]
```

**Full**
```
X_full = X_baseline
       + [CPI_qoq, CPI_yoy, expected_inflation]
       + [infl_shock_ma, infl_shock_ma_lag1]
       + [exp_shock_ma, exp_shock_ma_lag1]
```

## 3. 결과 해석 문장 템플릿

### A. Baseline 결과

> Baseline 모델은 lag1_비중을 중심으로 높은 설명력을 보였다. 이는 디저트 소비 비중이 강한 자기상관 구조를 가지며, 단기적으로 관성적 특성이 강함을 의미한다.

---

### B. Full 모델 (성능 개선 있음)

> 물가 및 충격 변수를 추가한 Full 모델은 Baseline 대비 RMSE가 감소하였다. 이는 인플레이션 변수들이 디저트 소비 변화에 추가적인 설명력을 제공함을 시사한다.

---

### C. Full 모델 (성능 개선 미미)

> 물가 변수를 포함하였으나 Baseline 대비 예측 성능 개선은 제한적이었다. 이는 디저트 소비가 거시적 물가 수준보다 지역 고유 특성과 소비 관성에 더 크게 영향을 받는다는 점을 보여준다.

---

### D. Shock 계수 유의한 경우

> 인플레이션 충격 계수(β)는 통계적으로 유의하였으며, 물가 상승 충격 발생 시 디저트 비중이 단기적으로 감소(또는 증가)하는 경향을 보였다.

---

### E. Shock 유의하지 않은 경우

> 물가 충격 변수는 통계적으로 유의하지 않았다. 이는 디저트 소비가 단기 물가 변동에 즉각적으로 반응하지 않는 구조적 특성을 가질 가능성을 시사한다.

---

### F. 군집별 해석

> 성장형 상권에서 인플레이션 충격에 대한 반응이 상대적으로 크게 나타났으며, 이는 상권 특성에 따라 물가 민감도가 상이함을 의미한다.

---

## 4. 보고서 결론 구조 템플릿

### ① 요약

- 디저트 소비는 강한 자기상관 구조를 보인다.
- 물가 변수는 동행성은 존재하나, 예측 기여는 제한적/부분적이다.
- 일부 군집에서는 물가 충격에 대한 반응이 관찰된다.

### ② 경제적 해석

- 디저트는 습관재(habit good) 성격이 강함
- 단기 인플레이션 충격은 제한적 영향
- 상권 구조가 더 중요

### ③ 한계

- 분기 데이터 → 표본 수 제한
- 전국 단위 물가 → 지역 이질성 반영 한계
- 코로나 효과 완전 통제 어려움
