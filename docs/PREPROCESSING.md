# 전처리 컬럼 설계

## 추가된 ML용 변수

| 컬럼 | 설명 | 용도 |
|------|------|------|
| `디저트_비중` | (카페+제과점 매출) / 전체 상권 매출 | 선호도 지표, 규모 보정 |
| `log_당월_매출_금액` | log(매출+1) | 분산 안정화, skew 완화 |
| `lag1` | 전분기 매출 | 시계열 패턴 |
| `lag4` | 전년 동분기 매출 | 계절성, YoY 비교 |
| `성장률` | (이번분기 - 전분기) / 전분기 | 트렌드, 물가와 비교 |
| `month_sin` | sin(2π×월/12) | 계절성 인코딩 |
| `month_cos` | cos(2π×월/12) | 계절성 인코딩 |

## 사용 순서 (교수님 추천)

1. 비율화 → 2. 로그 변환 → 3. Lag 생성 → 4. 계절 변수 → 5. 다중공선성 확인

## 실행

```bash
python scripts/run_preprocess.py
```
